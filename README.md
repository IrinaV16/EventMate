# EventMate

EventMate is a web application developed with Django that helps people organize events, discover activities, and connect with others who share similar interests. Users can create events, apply to participate, communicate through a real-time chat, leave reviews, and build a network of friends.

---

## Features

- User registration, login, and authentication
- User profiles with profile picture, bio, location, and interests
- Create, edit, and delete events
- Browse and search events by title or location
- Filter events by category
- Apply to participate in events
- Accept or reject participant applications
- Friends system with friend requests
- Real-time event chat using WebSockets
- Event reviews and ratings
- Notifications for important actions
- Weather forecast for events using the Open-Meteo API
- Event location preview with OpenStreetMap

## Project Structure

The project is divided into several Django applications:

### Accounts
Handles user authentication, profiles, dashboard, and profile management.

### Events
Manages event creation, editing, deletion, searching, filtering, and event details.

### Applications
Allows users to apply for events and enables organizers to accept or reject applications.

### Friends
Implements the friend request system and friend management.

### Chat
Provides real-time communication between event organizers and accepted participants using Django Channels and WebSockets.

### Reviews
Allows accepted participants to leave ratings and reviews after an event has finished.

### Notifications
Stores and displays notifications generated by important user actions.

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd EventMate
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment.

Install the required packages:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the development server:

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```

## APIs Used

### Open-Meteo API

Used to retrieve weather forecasts for event locations.

The application first converts the entered location into geographical coordinates using the Open-Meteo Geocoding API and then retrieves the weather forecast for the selected event date.

### OpenStreetMap

Provides a direct link to the event location without requiring an API key.
