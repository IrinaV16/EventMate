# EventMate

EventMate is a web application that helps people organize events and connect with others who share similar interests. Users can create activities, apply to participate, communicate with other participants, and build new friendships through the platform.

The goal of the project is to provide a simple and user-friendly way to organize social activities while encouraging communication between participants before and after each event.

---

## Main Features

### User Accounts

Users can create an account, log in securely, edit their profile, upload a profile picture, and add personal information such as a biography, location, and interests.

### Event Management

Registered users can create their own events by providing a title, description, location, category, date, and maximum number of participants.

Event organizers can edit or delete their own events at any time.

### Event Applications

Users can browse available events and apply to join them.

The organizer can review all applications and decide whether to accept or reject each participant.

Applications are automatically prevented when:
- the event has already taken place;
- the event is full;
- the user has already applied.

### Friends System

Users can send, accept, or reject friend requests.

Accepted friends can easily access each other's profiles through the application.

### Event Chat

Each event has its own private chat room.

Only the event organizer and accepted participants can access the chat, allowing them to communicate in real time before the event.

### Reviews

After an event has finished, accepted participants can leave a rating and a review for the organizer.

Each participant can submit only one review per event.

### Notifications

The system automatically generates notifications for important actions, including:

- new event applications;
- accepted or rejected applications;
- friend requests;
- accepted friend requests;
- newly submitted reviews.

### Weather Forecast

Each event page displays a weather forecast for the selected location using the Open-Meteo API.

The forecast is available for events taking place within the next 16 days.

### Event Location

Every event includes a direct link to OpenStreetMap, allowing users to quickly view the event location.
